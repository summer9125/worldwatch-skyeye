## 📖 必读书籍

### 人机交互

1. **《Don't Make Me Think》** - Steve Krug
   - 可用性设计经典，必读

2. **《The Design of Everyday Things》** - Don Norman
   - 设计心理学基础

3. **《About Face: The Essentials of Interaction Design》** - Alan Cooper
   - 交互设计圣经

4. **《Universal Principles of Design》** - William Lidwell
   - 设计原则百科全书

### 视觉设计

5. **《Thinking with Type》** - Ellen Lupton
   - 排版设计基础

6. **《Grid Systems in Graphic Design》** - Josef Müller-Brockmann
   - 网格系统设计

7. **《Interaction of Color》** - Josef Albers
   - 色彩理论经典

### 数据可视化

8. **《The Visual Display of Quantitative Information》** - Edward Tufte
   - 数据可视化圣经

9. **《Information Dashboard Design》** - Stephen Few
   - 仪表盘设计指南

10. **《Storytelling with Data》** - Cole Nussbaumer Knaflic
    - 用数据讲故事

### 用户体验

11. **《Lean UX》** - Jeff Gothelf
    - 敏捷 UX 设计

12. **《Sprint》** - Jake Knapp
    - Google 设计冲刺方法

13. **《Hooked》** - Nir Eyal
    - 用户习惯养成设计

## 📱 移动端适配最佳实践

### 响应式断点

```css
/* 手机 */
@media (max-width: 640px) { }

/* 平板 */
@media (min-width: 641px) and (max-width: 1024px) { }

/* 桌面 */
@media (min-width: 1025px) { }
```

### 触控优化

- 按钮最小尺寸：44x44px (Apple HIG)
- 点击区域间距：至少 8px
- 避免 hover 状态（手机不支持）
- 使用 `touch-action` 优化滚动

### 字体大小

- 正文：至少 16px
- 小字：至少 14px
- 标题：层次分明（24px / 20px / 16px）

### 安全区域

```css
/* iPhone 安全区域 */
padding: env(safe-area-inset-top) env(safe-area-inset-right) 
         env(safe-area-inset-bottom) env(safe-area-inset-left);
```

## 🎨 设计原则检查清单

### 可用性

- [ ] 对比度足够（WCAG AA 标准）
- [ ] 字体大小可读
- [ ] 点击区域够大
- [ ] 加载状态明确
- [ ] 错误提示清晰

### 视觉层次

- [ ] 重要内容突出
- [ ] 间距一致（8px 网格）
- [ ] 颜色使用克制（主色 + 辅助色）
- [ ] 字体不超过 3 种

### 交互反馈

- [ ] 点击有反馈
- [ ] 加载有动画
- [ ] 成功/错误有提示
- [ ] 状态变化明显

### 性能

- [ ] 首屏加载 < 3 秒
- [ ] 图片优化
- [ ] 减少动画复杂度
- [ ] 懒加载内容

## 📊 仪表盘设计要点

### 数据展示

1. **重要指标放顶部** - 用户第一眼看到
2. **使用合适的图表** - 趋势用折线，占比用饼图
3. **避免信息过载** - 一屏显示核心信息
4. **提供上下文** - 数字需要对比才有意义

### 颜色使用

- **成功/正常**: 绿色 (#00ff88)
- **警告/中等**: 黄色/橙色 (#ffa502)
- **危险/严重**: 红色 (#ff4757)
- **信息/普通**: 蓝色 (#00d9ff)

### 布局模式

```
┌─────────────────────────────────────┐
│  Header (Logo + Status + Time)      │
├──────────┬──────────────┬───────────┤
│  Stats   │    Main      │  Alerts   │
│  Cards   │   Content    │   List    │
├──────────   (Chart/    ├───────────┤
│  Filter  │    Map)      │  Details  │
│  Panel   │              │   Panel   │
└────────────────────────┴───────────┘
```

## 📝 设计评审清单

每次设计更新前检查：

1. **视觉**
   - 配色是否和谐？
   - 字体大小是否合适？
   - 间距是否一致？

2. **交互**
   - 操作是否直观？
   - 反馈是否及时？
   - 错误处理是否友好？

3. **性能**
   - 加载速度如何？
   - 动画是否流畅？
   - 是否支持离线？

4. **可访问性**
   - 色盲用户能否使用？
   - 字体是否够大？
   - 对比度是否足够？
