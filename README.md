## 🎯 系统定位

**全球时事热点情报系统** - 为 WaveUP 提供全球视野和情报支持

---

## 📊 数据源

### 1. World Monitor
**网址：** https://www.worldmonitor.app/
**功能：** 实时监控全球事件
**数据类型：**
- 国际新闻
- 政治事件
- 经济动态
- 社会热点
- 自然灾害

### 2. Cloudflare Radar
**网址：** https://radar.cloudflare.com/
**功能：** 互联网流量和攻击监控
**数据类型：**
- 网络攻击趋势
- 互联网中断事件
- 流量异常监控
- 地缘技术冲突

## 🎯 核心功能

### 1. 全球热点监控
- 实时采集 World Monitor 数据
- 按地区、类型分类
- 热度排序和趋势分析

### 2. 网络情报监控
- Cloudflare 网络攻击监控
- 互联网中断事件
- 地缘技术冲突

### 3. 情报分析
- 事件关联分析
- 影响范围评估
- 趋势预测

### 4. 情报简报
- 每日情报汇总
- 每周趋势报告
- 重大事件专报

### 5. 告警通知
- 重大事件实时告警
- 飞书/钉钉通知
- 告警级别分类

## 💡 与 WaveUP 舆情系统的关系

```
┌──────────────────┐         ┌──────────────────┐
│   天眼 SkyEye    │────────→│  WaveUP 舆情系统 │
│  (全球情报)      │  情报   │  (品牌舆情)      │
│                  │  输入   │                  │
└──────────────────┘         └──────────────────┘
```

**天眼提供：**
- 全球热点事件
- 国际舆情背景
- 潜在风险预警

**舆情系统利用：**
- 结合全球事件分析品牌舆情
- 预判可能的品牌影响
- 制定应对策略
